# JeetWeb Landing (Client-side React)

This is a lightweight Vite + React landing page for JeetWeb with smooth animations using Framer Motion.

Quick start:

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

Then open the local dev URL shown by Vite.

You can build with `npm run build` and preview the production build with `npm run preview`.
